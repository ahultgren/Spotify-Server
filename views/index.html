Frodj

tl;dr

Frodj is a Spotify app for all those occasions when everyone wants a say in what song is played. Let your friends choose what’s played through a web interface, or retain all the power yourself from the comfort of your couch.

How to use Frodj

Currently Frodj is in the early stage of development. Most importantly this means that Spotify haven't approved the app yet and so it's not available from the App Finder. Thus, if you want to use Frodj now you have to use Spotify Premium, get a developer account on Spotify and download the app from github. It's all explained step by step in the repo for Spotify Slave.

It also means that I haven't built all the features yet and that the UI is a bit unpolished. However that only means that I'm very open for suggestions, ideas and feedback about where to take Frodj. If you have an awesome feature in mind and want to be part of the development of Frodj, don't hesitate to create an issue on github or send a tweet my way.

Current features

  • See what track is played right now (track name, artist, album, duration)
  • See various player states (position, volume, whether shuffling and/or repeating)
  • Control playback (play/pause, next, previous)
  • Search for songs via Spotify's web API and play/or queue them
  • Admin mode, which prevents not logged in users from directly manipulating playback

Planned features

  • Display play queue
  • Vote to change track
  • Vote to order queued tracks
  • Chat with admin or other listeners
  • More fine-grained permissions system
