Spotify Server

Let the whole team choose what's played in the office speakers. Also works if you want people at say a party easily check what song is currently being played, if they're too shy to ask someone.

How to use Spotify Server

/play
Plays current song if no song is playing, pauses if a song is currenty playing.

/next
Plays next song in the play queue.

/prev
Plays previous song in the play queue.

/get/property
Get a player or track property. Currently supported properties: /state, /position, /volume, /name, /artist, /album, /duration, /url.

/set/property/value
Set a player property. Currently supported:

  • /state - play or pause.
  • /position - Number of seconds. Must be at least 0 and less than the lenght of the track.
  • /volume - Number of seconds. Must be between 0 and 100.
